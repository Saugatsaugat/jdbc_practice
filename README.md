# Mini Project - "Library Management System"
## Features:
    - Track books : title, author, category, quantity, added_date.
    - Tables: users, books, usersbooks
 * Operations:
    - CRUD for users, books, usersbooks
    - Issue/return a book
    - All list(books, users, records)
    - Reports by users, books etc.
 * Implementation:
    - Use plain JDBC for practice.
